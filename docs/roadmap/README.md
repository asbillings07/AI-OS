# Roadmap

> Primary question: **How will the product evolve over time?**

This folder describes Orion's planned evolution — what we're building now, what comes next, and (just as importantly) what we're deliberately *not* doing yet. The roadmap moves faster than the vision and principles, but every milestone should serve them.

## Current

- **[MVP Definition — v0.1](./mvp.md)** (#6) — Gmail-only, advisory, single dashboard. The deliberately tiny first slice, with an explicit "no" list.

## After v0.1 (candidates, not commitments)

Drawn from the MVP's "deliberately deferred" section: a second source (likely Calendar or GitHub), cross-source context, learning from user corrections, richer notifications, and the first *agentic* action behind a deterministic gate.
